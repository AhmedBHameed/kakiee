# Notification component

Use this component to generate system notification for the user.

# Notification example

<Preview withToolbar>
  <Story name="Notification">
    <Notification
      message={"This is a test notification message."}
      variant={"success"}
      open={true}
      direction={{
        vertical: "bottom",
        horizontal: "right"
      }}
    />
  </Story>
</Preview>

## Props values

| Prop name | Type    | Default value                               |
| --------- | ------- | ------------------------------------------- |
| message   | String  | "No message provided!"                      |
| variant   | String  | "success"                                   |
| open      | Boolean | false                                       |
| direction | Object  | { vertical: "bottom", horizontal: "right" } |
